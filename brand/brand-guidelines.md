# kafkade — Brand Guidelines

## Origin

**kafkade** = Kafka (Franz Kafka, the author) + Cascade (the mountain range).
A name for a software creator who builds at the intersection of thoughtful writing and technical craft.

## Color System

### Primary Palette (RockON!)

| Role       | Name      | Hex       | RGB                | Usage                                   |
|-----------|-----------|-----------|--------------------|-----------------------------------------|
| Deep      | Midnight  | `#010221` | `1, 2, 33`        | Primary background, hero sections       |
| Primary   | Teal      | `#0A7373` | `10, 114, 114`    | Links, accents, interactive elements    |
| Neutral   | Sage      | `#B7BF99` | `182, 191, 153`   | Secondary text, borders, muted UI       |
| Accent    | Amber     | `#EDAA25` | `237, 169, 36`    | CTAs, highlights, hover states          |
| Danger    | Ember     | `#C43302` | `196, 50, 1`      | Warnings, emphasis, destructive actions |

### Extended Palette

| Role              | Hex       | Usage                              |
|-------------------|-----------|-------------------------------------|
| Surface Light     | `#F5F5F0` | Light mode background               |
| Surface Dark      | `#020318` | Card backgrounds on dark mode       |
| Text Light Mode   | `#1A1A2E` | Primary text (light mode)           |
| Text Dark Mode    | `#E8E8E0` | Primary text (dark mode)            |
| Teal Light        | `#0D9E9E` | Hover state for teal elements       |
| Amber Light       | `#F5C04A` | Hover state for amber elements      |

### Usage Rules

- **Dark mode is default**. The palette was designed dark-first.
- Never use pure black (`#000`) or pure white (`#FFF`) for text or backgrounds.
- Teal is the primary interactive color. Use it for links, buttons, and focus states.
- Amber is reserved for emphasis — don't overuse it.
- Ember is for destructive actions and critical emphasis only.
- Sage works for borders, dividers, and secondary/muted text.

## Typography

| Role      | Font                | Fallback              | Weights    | Usage                        |
|-----------|--------------------|-----------------------|------------|------------------------------|
| Display   | Playfair Display   | Georgia, serif        | 700, 900   | Headings, hero text          |
| Body      | Inter              | system-ui, sans-serif | 400, 500   | Body text, UI elements       |
| Mono      | JetBrains Mono     | monospace             | 400        | Code blocks, technical text  |

### Type Scale

```
Display:    3rem / 48px  — Page titles, hero
H1:         2.25rem / 36px
H2:         1.5rem / 24px
H3:         1.25rem / 20px
Body:       1rem / 16px
Small:      0.875rem / 14px
Caption:    0.75rem / 12px
```

### Rules

- Headings always use Playfair Display. Body always uses Inter.
- Line height: 1.6 for body, 1.2 for display/headings.
- Max content width: 720px for readability.
- Paragraph spacing: 1.5em.

## Logo

### Wordmark (`logo-full.svg`)
The primary logo. "kafkade" rendered as a clean SVG wordmark. The stylized letterforms use Midnight and Teal.

### Monogram (`logo-monogram.svg`)
A "K" letterform inside a circle. Used where space is limited.

### Favicon (`favicon.svg`)
The monogram adapted for 16×16 and 32×32 display.

### Avatar (`avatar.png`)
512×512 version of the monogram for social profiles.

### Usage Rules

- Always use the SVG version when possible.
- Minimum clear space: 1× the height of the "k" on all sides.
- On dark backgrounds: use Teal or E8E8E0 (warm white) for the mark.
- On light backgrounds: use Midnight for the mark.
- Never stretch, rotate, or add effects to the logo.

## Voice & Tone

### Personality
- **Technical but approachable** — explain complex things simply
- **Thoughtful** — take time to consider, don't rush to opinions
- **Literary undertones** — a builder who reads
- **Honest** — share what works and what doesn't

### Tagline
> Code. Craft. Cascade.

### Writing Guidelines
- Prefer active voice.
- Keep sentences concise but not terse.
- Use technical terms accurately, but explain them when writing for a general audience.
- It's OK to have personality — this isn't a corporation.
